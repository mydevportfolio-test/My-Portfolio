Skills

Atmos

Atmos is a full-stack application for controlling Philips Hue smart lights. It aims to fill the void in desktop applications available for these smart lights.

Technology

Atmos is built on React, Express, and Node.js for real-time light control. Integrating with the Philips Hue API for light control. A Node.js server hosted locally acts as a middle-man, handling backend logic and database updates within PostgreSQL.
